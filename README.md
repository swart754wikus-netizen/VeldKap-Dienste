# Veldkap Dienste

Landing page for Veldkap Dienste — a lawn equipment retailer.

## Structure
- `index.html` — page markup (hero, products, why-us, contact)
- `styles.css` — styling
- `script.js` — small runtime touches (footer year)

## Deploy
Static site, no build step. Deploy by dragging the folder into [Netlify Drop](https://app.netlify.com/drop), or connect this repo to Netlify/Vercel/GitHub Pages.
